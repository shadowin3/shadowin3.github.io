Message Deletion

Can be used to quickly delete any desired amount of messages from the text channel.

  • Keyword
  • Note
  • Example
Delete, Prune, Purge, Clear, Clean
This command requires you to specify the number of messages you wish deleted from the server. Please keep in mind if you wish to do a specific amount that you add 3 to the number you wish to delete, as the number includes the messages used to enact the command.
Hey Echo, please clear 5 messages from the channel.
Kicking Members

Can be used to temporarily remove a member from your server.

  • Keyword
  • Note
  • Example
Kick
This command requires you to mention the person that you wish removed from the server, as the punishment is enacted via their User ID.
Echo, please kick @JohnSmith#1234
Muting/Unmuting Members

Can be used to mute or unmute members, removing or reinstituting their ability to speak in text channels.

  • Keywords
  • Note
  • Example
Mute, Unmute
This command can currently be used only in Echo's home server, as it involves roles which go by a specific ID. Additionally, this command requires you to mention the person you wish muted, as it adds the role to the user via their ID.
  • Ok Echo, mute @JohnSmith#1234
  • @Echo#2584, unmute @JohnSmith#1234
Banning/Unbanning Members

Can be used to permanently remove someone from your server.

  • Keywords
  • Note
  • Example
Ban, Unban
This command requires you to mention the member you wish to ban, as it enacts the punishment via their user ID. For unbanning, you must give Echo the user ID of the person you wish to be unbanned. To find this, right click on the tab for private messages with the user, and click the words: "copy ID."
  • @Echo#2584, ban @JohnSmith1234
  • Unban 123412341234123412, Echo
